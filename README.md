Plataforma de Serviços Locais

Uma plataforma web funcional desenvolvida para conectar moradores de bairros a profissionais locais, facilitando a economia circular. O design foi totalmente baseado em um protótipo do Figma e implementado com tecnologias modernas de desenvolvimento web.

Funcionalidades
Busca Inteligente: Filtre profissionais por nome, tipo de serviço ou bairro.
Cards Interativos: Visualização clara com ícones, especialidades e localização.
Integração direta com WhatsApp:** Clique no botão para abrir uma conversa já com uma mensagem personalizada para o profissional.

Tecnologias Utilizadas
Linguagem: Python
Framework Web: Flask
Estilização: Tailwind CSS (via CDN)
Design: Figma

Prévia do Design
O projeto foi construído para ser "pixel perfect" em relação ao design original, garantindo uma interface limpa, moderna e responsiva.
